

## 📝 프로젝트 개요
- 은행 마케팅-관리를 위한 고객의 '연체 위험도' 예측 머신러닝 모델 구현 <br>
- 사용 데이터 : 2만6천여명 고객의 성별, 부동산, 자동차, 가족, 등록기간 등의 정보

## 새롭게 배운 것
- 모델링 전반의 과정
- EDA를 꼼꼼히 할수록 모델의 성능이 개선된다
- 환경 구축은 중요하기도 하고, 시간이 많이 걸린다
- 모델 구현 뿐 아니라, 해석과 기대효과 부분도 매우 중요하다
